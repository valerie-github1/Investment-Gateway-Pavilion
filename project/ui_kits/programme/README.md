# Programme UI Kit — WIS Abuja 2026

A faithful recreation of the **Official Programme** for the World Press Conference. Dark surface; deep near-black with two large border-only orbital rings; ceremonial gold accents.

## Components
| File | What |
|---|---|
| `ProgrammeHero.jsx` | Dark hero — orbital rings (40s + 28s), logo crest, eyebrow, italic display title. |
| `VenueCard.jsx` | Two-cell venue & convener card with hairline divider. |
| `DaySection.jsx` | Day timeline — large gold serif day numeral, italic title, time/title/desc/speaker rows with hover gradient rail. |
| `ProgrammeFooter.jsx` | Centered crest + tracked Cinzel logotype. |

## Source
Recreated against `WIS_Abuja_2026_Elegant_Programme.html` from the GitHub repo `valerie-github1/wis.abuja.2026`.

## Caveats
- Day-tab navigation is added for convenience; the source document is a single scrolling printable. Remove tabs for print.
- Speaker names are placeholder titles — replace with real schedule when finalised.
